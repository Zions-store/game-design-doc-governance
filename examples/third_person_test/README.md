# ThirdPersonTest Example Project

This is the real-world project that the `game-design-doc-governance` Skill
was extracted from — an open-world narrative tactical shooter (Unreal Engine 5.8).

Use this example to understand how a project profile extends a genre profile
with project-specific fact checks, language rules, and boundary checks.

## Files

- `Project_Profile.yaml` — Project instance profile (extends the open_world genre)
- `expected_baseline.json` — Expected audit baseline (P0=0, P1=0, P2=0, P3=1)
